# e-plantShopping

## Project Overview
**e-plantShopping** is a simple React-based plant shopping application that allows users to browse available plants, view product details, and add items to a cart for checkout. The goal of this project is to demonstrate core front-end development skills such as component-based UI design, state management, and user interaction handling.

## Features
- Browse a list of plants/products
- View basic product information
- Add and remove items from the shopping cart
- Responsive layout for better usability

## Tech Stack
- React
- JavaScript (ES6)
- HTML/CSS
- Vite (development build tool)
